# AWS-Serverless-Cost-Optimization-Practice
This project demonstrates a basic serverless setup using AWS Lambda integrated with API Gateway, along with practical cost optimization techniques implemented under the AWS Free Tier.
What I Implemented
Integrated AWS Lambda with API Gateway (HTTP API)
Monitored Lambda execution using Amazon CloudWatch logs and metrics
Implemented Amazon S3 lifecycle policies to reduce storage costs
Reviewed AWS Trusted Advisor cost optimization checks available with Basic support
AWS Services Used
AWS Lambda
API Gateway
Amazon CloudWatch
Amazon S3
AWS Trusted Advisor
Key Learnings
Full Trusted Advisor checks require Business or Enterprise support plans.
Under Free Tier, effective cost optimization is achieved through lifecycle policies, monitoring execution metrics, and avoiding idle resources.
